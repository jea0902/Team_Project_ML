## <프로젝트 내용>
1. ML
    * 데이터 파악
    * 프로젝트 목표 설정
    * 데이터 전처리
    * 데이터 시각화
    * 데이터 인코딩
    * 모델 학습
    * 모델 평가, 검증

2. 예측 모델
   * 로지스틱 회귀
   * 랜덤포레스트
   * 그래디언트 부스팅
   * XGBoost
   * SVC
   * 의사결정트리
   * 라이트GBM

    위 모델들을 ROC곡선, 기능 중요도에 대한 SHAP,
    순열, Feature 중요도로 검증,평가하여

    과적합 규제 기능이 강하고,
    분류와 회귀영역에서 뛰어난 예측 성능을 가졌기에
    XGBoost를 최종 모델로 선정
   
3. 솔루션 제안
   * 도출된 인사이트들로 솔루션 제안
